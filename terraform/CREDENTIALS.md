# Credential Management Guide\n\n## Setup\n\n1. Copy the example files:\n   ```bash\n   cp .env.example .env\n   cp secrets.tfvars.example secrets.tfvars\n   ```\n\n2. Edit the files with your actual credentials:\n   - `.env` - for environment variables\n   - `secrets.tfvars` - for Terraform variables\n\n## Usage\n\n### Method 1: Environment Variables\n```bash\n# Load environment variables\nsource .env\nterraform apply -var-file=\"testing.tfvars\"\n```\n\n### Method 2: Secrets tfvars file\n```bash\nterraform apply -var-file=\"testing.tfvars\" -var-file=\"secrets.tfvars\"\n```\n\n### Method 3: Command line (not recommended)\n```bash\nterraform apply -var=\"ftp_pass=your_password\" -var-file=\"testing.tfvars\"\n```\n\n## Security Notes\n\n- Never commit `.env` or `secrets.tfvars` files\n- Use AWS IAM roles instead of access keys when possible\n- Rotate credentials regularly\n- Consider using AWS Secrets Manager for production
